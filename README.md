# Localstack sandbox

[Localstack docs](https://docs.localstack.cloud/overview/)

[Localstack GitHub](https://github.com/localstack/localstack/)


## Starting localstack

In interactive mode 
> cd localstack && docker compose up

In detached mode
> cd localstack && docker compose up -d
